Sudoku and sudoku solver with backtracking algorithm used when solving, Ruby

Run game.rb
Type the txt-filename without the file extension or quotations. (e.g. sudoku2)
Type "solve" without quotations to solve the current maze with backtracking algorithm.

Create your own maze sudoku:
- Create a 9x9 txt-file
- Zeroes are unassigned and numbers are preassigned
- The program will automatically detect your preassigned numbers and make them immutable
